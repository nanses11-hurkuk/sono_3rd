## 1. 메인/메뉴 구성

### 1.1. 메인 화면

메인 화면은 다음의 주요 영역으로 구성됩니다:

| 영역 | 설명 |
|------|------|
| #1 | 전자결재 좌측 메뉴 |
| #2 | 전자결재 포탈 |
| #2-1 | 타임라인: 결재 상태와 결재정보를 보여주는 알림 메시지 |
| #2-2 | 결재현황: 읽지 않은 회람문서, 읽지 않은 참조 문서, 읽지 않은 미결문서의 각 건수와 목록 조회 |
| #2-3 | 양식 즐겨 찾기 목록 조회 |

### 1.2. 메뉴

전자결재 메뉴는 다음과 같이 구성됩니다:

#### 1. 결재 작성 버튼
클릭 시 결재 작성 페이지로 이동

#### 2. 개인함
- **2-1 예고함:** 사용자가 결재하게 될 문서함 (결재 순서가 아직 오지 않은 문서)
- **2-2 미결함:** 사용자가 결재할 문서함 (결재 순서가 사용자 차례인 문서)
- **2-3 진행함:** 사용자가 결재를 작성하여 완료되지 않은 문서나 사용자가 결재선에 포함되어 있으나 완료되지 않은 문서
- **2-4 완료함:** 결재 진행이 완료된 문서
- **2-5 반려함:** 결재 진행이 반려된 문서
- **2-6 참조/회람함:** 사용자에게 참조/회람된 문서
- **2-7 임시함:** 사용자가 작성 중 임시 저장한 문서

#### 3. 부서함
- **3-1 완료함:** 부서 내 완료된 문서
- **3-2 발신함:** 부서 내 발신 문서
- **3-3 수신함:** 부서 내 수신 문서
- **3-4 수신진행함:** 부서 내 진행 중인 문서
- **3-5 수신처리함:** 부서로 수신된 문서 중에서 재기안 처리가 되어 결재가 완료된 문서
- **3-6 참조/회람함:** 부서 내 참조/회람 문서

#### 4. 설정
전자결재 환경 설정

#### 5. 개인폴더
개인적으로 사용할 폴더를 등록, 관리

#### 6. 담당업무함
사용자가 담당 업무자로 지정된 결재 문서 (사용자가 승인/반려 해야함)

#### 7. 업무문서함
사용자가 업무자로 지정된 결재 문서 (승인/반려 기능은 없고 열람 가능)

## 2. 결재작성

### 2.1. 결재작성(양식함)

전자결재에서 시작되는 양식 목록을 표시합니다. 최근기안한 양식, 즐겨찾기 설정한 양식, 분류/검색에 따른 양식을 조회할 수 있습니다.

| 항목 | 설명 |
|------|------|
| #1 | 결재 작성 가능한 양식 목록이 표시됨 |
| #2 | 최근 기안한 양식 목록을 조회하여 #1에 표시 |
| #3 | 즐겨찾기 설정한 양식 목록을 조회하여 #1에 표시 |
| #4 | 입력된 검색어를 기준으로 양식 목록을 조회하여 #1에 표시 |
| #5 | 결재양식에 대한 분류를 표시. 항목 클릭 시 분류에 해당하는 결재양식 목록을 조회하여 #1에 표시 |
| #6 | 항목 클릭 시 결재문서 작성화면을 새 창으로 표시 |
| #7 | 즐겨찾기 설정/해제 |
