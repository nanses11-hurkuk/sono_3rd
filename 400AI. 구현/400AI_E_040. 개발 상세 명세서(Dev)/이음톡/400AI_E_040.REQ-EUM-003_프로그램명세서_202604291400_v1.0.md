**EumI18nMapper.java** — MyBatis Mapper 인터페이스:
```java
@Mapper
public interface EumI18nMapper {
    // 특정 언어의 전체 메시지 키-밸류 목록 조회
    List<I18nMessage> findAllByLang(@Param("lang") String lang);
    // 특정 키의 베트남어 메시지 조회

```

**EumI18nMapper.xml**:
```xml
<mapper namespace="com.eum.i18n.mapper.EumI18nMapper">

  <select id="findAllByLang" resultType="map">
    SELECT MSG_KEY,
           CASE #{lang}
               WHEN 'vi' THEN MSG_VI
               WHEN 'en' THEN MSG_EN
               ELSE MSG_KO
           END AS MSG_VALUE
      FROM SYS_MULTILANGUAGE
     WHERE CASE #{lang}
               WHEN 'vi' THEN MSG_VI
               WHEN 'en' THEN MSG_EN
               ELSE MSG_KO
           END IS NOT NULL
  </select>

  <select id="findViMessageByKey" resultType="String">
    SELECT MSG_VI
      FROM SYS_MULTILANGUAGE
     WHERE MSG_KEY = #{msgKey}
  </select>

</mapper>
```

#### 캐시 설계 — Redis 언어 리소스 캐시

| 항목 | 내용 |
|------|------|
| **저장소** | Redis |
| **키 패턴** | `eum:i18n:{lang}` |
| **키 예시** | `eum:i18n:vi` |
| **값 형식** | JSON 문자열 (Map<String, String> 직렬화) |
| **값 구조 예시** | `{"chat.send":"Gửi","chat.receive":"Nhận","chat.room.create":"Tạo phòng chat",...}` |
| **TTL** | 무제한 (0). 언어 리소스는 배포 시점에 갱신되므로 TTL 기반 만료 대신 배포 이벤트 기반 무효화 적용. |
| **캐시 무효화** | 배포 완료 직후 `DEL eum:i18n:vi` 실행 (배포 스크립트에 포함). 또는 관리자 API `DELETE /api/admin/i18n/cache?lang=vi` 호출. |
| **캐시 미스 처리** | `SYS_MULTILANGUAGE` 테이블에서 해당 언어 전체 메시지 로드 후 Redis에 저장. 이후 요청은 캐시 히트 처리. |
| **네임스페이스 충돌 방지** | `eum:i18n:*` 패턴으로 기존 Redis 키와 분리. |

```java
@Component
public class EumLocaleResolver implements LocaleResolver {

    @Override
    public Locale resolveLocale(HttpServletRequest request) {
        // 우선순위 1: 사용자 세션에 저장된 로케일
        HttpSession session = request.getSession(false);
        if (session != null) {
            String sessionLocale = (String) session.getAttribute("USER_LOCALE");
            if (StringUtils.hasText(sessionLocale)) {
                return Locale.forLanguageTag(sessionLocale);
            }
        }
        // 우선순위 2: Accept-Language 헤더
        String acceptLang = request.getHeader("Accept-Language");
        if (StringUtils.hasText(acceptLang)) {
            Locale locale = Locale.forLanguageTag(acceptLang.split(",")[0].trim());
            if (isSupportedLocale(locale)) {
                return locale;
            }
        }
        // 우선순위 3: 시스템 기본값
        return Locale.KOREAN;  // ko
    }

    @Override
    public void setLocale(HttpServletRequest req, HttpServletResponse res, Locale locale) {
        if (locale != null && isSupportedLocale(locale)) {
            req.getSession(true).setAttribute("USER_LOCALE", locale.toLanguageTag());
        }
    }

    private boolean isSupportedLocale(Locale locale) {
        String lang = locale.getLanguage();
        return "vi".equals(lang) || "ko".equals(lang) || "en".equals(lang);
    }
}
```

**EumI18nService.loadMessages()**:
```java
@Service
public class EumI18nService {

    private final EumI18nMapper        i18nMapper;
    private final RedisTemplate<String,String> redisTemplate;
    private final ObjectMapper         objectMapper;

    private static final String CACHE_KEY_PREFIX = "eum:i18n:";

    public Map<String, String> loadMessages(String lang) {
        String cacheKey = CACHE_KEY_PREFIX + lang;

        // 1. Redis 캐시 우선 조회
        String cached = redisTemplate.opsForValue().get(cacheKey);
        if (StringUtils.hasText(cached)) {
            return objectMapper.readValue(cached, new TypeReference<Map<String,String>>(){});
        }

        // 2. 캐시 미스: DB 조회
        List<I18nMessage> rows = i18nMapper.findAllByLang(lang);
        if (rows == null || rows.isEmpty()) {
            log.warn("[EumI18nService] DB에서 '{}' 언어 리소스를 찾을 수 없습니다.", lang);
            return Collections.emptyMap();  // Fallback: 빈 맵 반환
        }

        // 3. Map 변환 및 Redis 캐시 저장 (TTL 무제한)
        Map<String,String> messages = rows.stream()
            .collect(Collectors.toMap(I18nMessage::getMsgKey, I18nMessage::getMsgValue));
        redisTemplate.opsForValue().set(cacheKey, objectMapper.writeValueAsString(messages));

        return messages;
    }

    // 캐시 무효화 (배포 이벤트 또는 관리자 API 호출 시)
    public void invalidateCache(String lang) {
        redisTemplate.delete(CACHE_KEY_PREFIX + lang);
        log.info("[EumI18nService] '{}' 언어 리소스 캐시를 무효화하였습니다.", lang);
    }
}
```

#### 트랜잭션 관리

- `EumUserLocaleMapper.updateLocale()` 사용자 언어 설정 저장은 단일 레코드 UPDATE이므로 Spring `@Transactional`(REQUIRED)으로 원자성을 보장한다.
- `EumI18nService.loadMessages()`의 DB 조회는 `@Transactional(readOnly = true)`로 처리하여 Lock 부하를 최소화한다.
- Redis 캐시 저장은 트랜잭션 외부에서 처리한다. DB 조회 성공 후 Redis 저장 실패 시 서비스는 계속 동작하며(Fallback: DB 재조회), 실패 로그를 기록한다.

#### 비동기 처리

- `GET /api/common/i18n/messages` 요청은 동기 처리한다. 응답 속도가 UX에 직접 영향을 미치므로 Redis 캐시를 통해 응답 시간을 최소화(목표: 50ms 이내)한다.
- 배포 후 캐시 무효화는 `@EventListener(ApplicationReadyEvent.class)`를 통해 애플리케이션 시작 완료 시점에 자동 실행한다.
