- **목적**: 그룹웨어-Local AD-M365 간의 계정 속성 및 패스워드 동기화 상태를 검증하고, AADC를 통한 최종 반영 여부를 확인함.
- **난이도**: 중
- **범위**: LDAPS 연동 -> AD 속성 갱신 -> AADC 트리거 -> M365(Entra ID) 반영 확인.

### 2. 개발자(담당자) 배정

### 5. 통합테스트 환경 및 인프라 구축
- **구성**: Local AD Domain Controller (Win2019) - Azure AD Connect - M365 테넌트.
- **특이사항**: Port 636 보안 연결 및 인증서 신뢰 설정 완료.

### 6. 통합테스트 데이터 전략

- **결과**: AD 및 M365 관리 센터 내 속성 일치 확인.

### 7. 통합테스트 상세 범위
- **UI ↔ API Layer**: 사용자 정보 수정 UI에서 저장 시 `RestUserCon`을 통한 AD 동기화 성공 확인.
- **Service ↔ Service**: `ADSyncSvc`에서 LDAP 트랜잭션 정상 처리 확인.
- **App ↔ Database**: 동기화 결과 로그(`AdSyncResultVO`) 정상 적재 확인.
- **Async Tasks**: AADC 동기화 주기(30분) 이후 M365 반영 상태 최종 점검.
