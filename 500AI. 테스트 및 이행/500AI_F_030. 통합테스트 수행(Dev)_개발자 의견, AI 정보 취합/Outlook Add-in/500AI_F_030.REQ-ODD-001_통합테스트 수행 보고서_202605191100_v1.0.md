## (3) 개발자 의견

CAUTK 파라미터 기반 라우팅 구조 및 CoviMSAPI(org/contact) 연동 레이어는 안정적으로 동작함. G-01(세션 재연결), G-05(Redis Lock), G-06(비동기 조직도) 신규 구현 항목도 WireMock 모킹 환경 기준 정상 동작 확인. 단, G-02(OAuth2) 구간은 Azure AD 앱 권한 의존성이 높아 실 연동 검증이 지연되고 있음 — 인프라 조치 최우선 요청. 비동기 조직도(G-06)의 경우 베트남 현지 네트워크 지연 환경에서 PENDING 유지 시간이 길어질 수 있으므로 Staging 단계에서 폴링 타임아웃 임계값 재협의 권고.
