## (2) 개발 통합테스트 결과 내용

### 2-1. 목적 및 범위

- **목적**: 이음톡 베트남어(`vi`) 로케일 적용 — `ResourceBundle` 연동, `SYS_MULTILANGUAGE` 베트남어 데이터 반영, Noto Sans VI 폰트 렌더링 시나리오별 통합 검증 결과를 PM 보고 및 Staging 이관 기준 자료로 기록
- **범위**: `Accept-Language: vi` 감지 → `GET /api/common/i18n/messages?lang=vi` → `SYS_MULTILANGUAGE` 조회 → JSON 반환 → UI 키-밸류 렌더링(Noto Sans VI) 전 구간. 로케일 전환·세션 유지·성조 문자 렌더링 포함. 소스 수정 불가로 설정·데이터 기반 검증.
- **제외**: 소스 수정 불가로 내부 i18n 프레임워크 코드 레벨 검증 제외
